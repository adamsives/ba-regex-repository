Purpose of this document

Need to find all email addresses in document? Perhaps you need to validate a given email address in a user input field. Try this...it's not perfect but should get you for 99.99% of cases.

Other things

From Wikipedia:
"The general format of an email address is local-part@domain, and a specific example is jsmith@example.com. An address consists of two parts. The part before the @ symbol (local-part) identifies the name of a mailbox. This is often the username of the recipient, e.g., jsmith."
Source:Wikipedia

There are exceptions to the rule however.

List of Valid Email Addresses this regex will match

email@example.com
firstname.lastname@example.com
email@subdomain.example.com
email@123.123.123.123
1234567890@example.com
email@example-one.com
_______@example.com
email@example.name
email@example.museum
email@example.co.jp
firstname-lastname@example.com

List of Valid Email Addresses this regex will NOT match

email@[123.123.123.123]
“email”@example.com
firstname+lastname@example.com

List of Strange but Valid Email Addresses this regex will NOT match

much.“more\ unusual”@example.com
very.unusual.“@”.unusual.com@example.com
very.“(),:;<>[]”.VERY.“very@\\ "very”.unusual@strange.example.com
